# Transports & MCP Client

## Transports Overview

| | stdio | Streamable HTTP | SSE |
|---|---|---|---|
| Use for | Claude Desktop, local | Production, remote | Legacy only |
| Default endpoint | stdin/stdout | `/mcp` | `/sse` |
| Multi-node | ✗ (one proc/client) | ✓ (`stateless_http=True`) | ✗ (sticky sessions) |
| Status | Stable | **Recommended** | Deprecated (v1.25+) |

---

## stdio Transport

### Server

```python
mcp = FastMCP("My Server")
mcp.run()                     # default = stdio
mcp.run(transport="stdio")    # explicit
```

### Claude Desktop Config

```json
{
  "mcpServers": {
    "my-server": {
      "command": "uv",
      "args": ["run", "--with", "mcp", "mcp", "run", "/abs/path/to/server.py"],
      "env": { "API_KEY": "secret" }
    }
  }
}
```

Config file locations:
- macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
- Windows: `%APPDATA%\Claude\claude_desktop_config.json`

Auto-generated by: `uv run mcp install server.py --name "My Server" -v API_KEY=secret`

## Streamable HTTP Transport

### Basic server

```python
mcp = FastMCP("App", stateless_http=True, json_response=True)
mcp.run(transport="streamable-http")   # http://0.0.0.0:8000/mcp
```

Customize host/port/path:
```python
mcp = FastMCP("App", host="127.0.0.1", port=9000, streamable_http_path="/api")
```

### ASGI multi-server mounting

```python
import contextlib
from starlette.applications import Starlette
from starlette.routing import Mount
from mcp.server.fastmcp import FastMCP

api  = FastMCP("API",  stateless_http=True, json_response=True)
chat = FastMCP("Chat", stateless_http=True, json_response=True)

# Mount at /api and /chat (not /api/mcp and /chat/mcp)
api.settings.streamable_http_path  = "/"
chat.settings.streamable_http_path = "/"

@contextlib.asynccontextmanager
async def lifespan(app: Starlette):
    async with contextlib.AsyncExitStack() as stack:
        await stack.enter_async_context(api.session_manager.run())
        await stack.enter_async_context(chat.session_manager.run())
        yield

app = Starlette(
    routes=[
        Mount("/api",  app=api.streamable_http_app()),
        Mount("/chat", app=chat.streamable_http_app()),
    ],
    lifespan=lifespan,
)
# Run: uvicorn app:app
```

> **Critical:** Always call `session_manager.run()` in the ASGI lifespan — forgetting it silently breaks the server.

### CORS (for browser clients)

```python
from starlette.middleware.cors import CORSMiddleware

app = CORSMiddleware(
    app,
    allow_origins=["https://myapp.com"],   # not "*" in production
    allow_methods=["GET", "POST", "DELETE"],
    expose_headers=["Mcp-Session-Id"],     # required for browser clients
)
```

### Register with Claude Code

```bash
claude mcp add --transport http my-server http://localhost:8000/mcp
```

## SSE Transport (Legacy)

```python
# Server
mcp.run(transport="sse")      # GET /sse, POST /messages
mcp.sse_app()                  # ASGI mount

# Client
from mcp.client.sse import sse_client
async with sse_client("http://localhost:8000/sse") as (r, w):
    async with ClientSession(r, w) as session:
        await session.initialize()
```

> Migrate to Streamable HTTP for new projects. SSE requires sticky load balancing.

## MCP Client — Full API

```python
from mcp import ClientSession, StdioServerParameters, types
from mcp.client.stdio import stdio_client
from mcp.client.streamable_http import streamable_http_client
from mcp.client.sse import sse_client
from mcp.shared.context import RequestContext
from pydantic import AnyUrl

# --- Transport selection ---
# stdio
params = StdioServerParameters(command="uv", args=["run", "server.py"], env={"KEY": "val"})
async with stdio_client(params) as (r, w): ...

# Streamable HTTP
async with streamable_http_client("http://localhost:8000/mcp") as (r, w, _): ...

# Streamable HTTP with custom headers (e.g., Bearer token)
async with streamable_http_client(
    "http://localhost:8000/mcp",
    headers={"Authorization": "Bearer my-token"},
) as (r, w, _): ...

# --- Session usage ---
async with ClientSession(r, w, sampling_callback=my_sampling_fn) as session:
    await session.initialize()               # MUST be first call

    # Tools
    tools   = await session.list_tools()
    result  = await session.call_tool("add", {"a": 1, "b": 2})
    print(result.content[0].text)           # unstructured
    print(result.structuredContent)         # structured dict (or None)
    print(result.isError)                   # True if tool raised

    # Resources
    resources  = await session.list_resources()
    templates  = await session.list_resource_templates()
    resource   = await session.read_resource(AnyUrl("config://settings"))
    print(resource.contents[0].text)

    # Prompts
    prompts = await session.list_prompts()
    prompt  = await session.get_prompt("review_code", {"code": "def f(): pass"})
    print(prompt.messages[0].content.text)

    # Completion suggestions
    from mcp.types import ResourceTemplateReference
    comp = await session.complete(
        ref=ResourceTemplateReference(type="ref/resource", uri="file://{name}"),
        argument={"name": "name", "value": "read"},
    )
```

### Sampling Callback

Required if the server calls `ctx.session.create_message()`:

```python
async def my_sampling_fn(
    ctx: RequestContext[ClientSession, None],
    params: types.CreateMessageRequestParams,
) -> types.CreateMessageResult:
    # Route to your LLM
    return types.CreateMessageResult(
        role="assistant",
        content=types.TextContent(type="text", text="[LLM response]"),
        model="claude-3-5-sonnet",
        stopReason="endTurn",
    )

async with ClientSession(r, w, sampling_callback=my_sampling_fn) as session:
    ...
```

### OAuth Client

For protected servers (OAuth 2.1):

```python
import httpx
from mcp.client.auth import OAuthClientProvider, TokenStorage
from mcp.shared.auth import OAuthClientMetadata, OAuthToken, OAuthClientInformationFull

class FileTokenStorage(TokenStorage):
    # Implement: get_tokens, set_tokens, get_client_info, set_client_info
    ...

auth = OAuthClientProvider(
    server_url="http://localhost:8000",
    client_metadata=OAuthClientMetadata(
        client_name="My Client",
        redirect_uris=[AnyUrl("http://localhost:3000/callback")],
        grant_types=["authorization_code", "refresh_token"],
        response_types=["code"],
        scope="read write",
    ),
    storage=FileTokenStorage(),
    redirect_handler=lambda url: print(f"Open: {url}"),
    callback_handler=lambda: (input("Paste callback URL code: "), None),
)

async with httpx.AsyncClient(auth=auth, follow_redirects=True) as http_client:
    async with streamable_http_client(
        "http://localhost:8000/mcp",
        http_client=http_client,
    ) as (r, w, _):
        async with ClientSession(r, w) as session:
            await session.initialize()
```

> Use a persistent `TokenStorage` implementation (file or secrets manager) — `InMemoryTokenStorage` loses tokens on restart.

## Client Pitfalls

- **Forgetting `await session.initialize()`** — all subsequent calls fail.
- **No `sampling_callback`** when server uses sampling — call hangs indefinitely.
- **`stdio_client` with a server that writes to stdout** — breaks protocol framing.
- **Not checking `result.isError`** — silent errors in tool calls.
- **SSE `sse_client` URL must point to `/sse`**, not root (`/`).
